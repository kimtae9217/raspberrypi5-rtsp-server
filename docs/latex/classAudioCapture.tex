\doxysubsection{Detailed Description}
Audio Stream Data를 캡처하는 클래스 

PCM 오디오 데이터를 캡처하고 Opus 인코딩을 위한 처리를 수행하는 클래스로, \mbox{\hyperlink{classDataCapture}{Data\+Capture}} 클래스를 상속받아 오디오 스트림 관리 기능을 구현합니다.
