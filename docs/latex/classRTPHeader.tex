\doxysubsection{Detailed Description}
패킹 설정을 원래대로 복구 

RTP 헤더를 정의하고 처리하는 클래스

RTP 패킷의 마커 비트(\+M bit)를 설정하는 메서드입니다.
\begin{DoxyItemize}
\item true로 설정 시\+: marker $\vert$= 0x01 (비트 OR 연산으로 마지막 비트를 1로 설정)
\item false로 설정 시\+: marker \&= $\sim$0x01 (비트 AND 연산으로 마지막 비트를 0으로 설정)
\end{DoxyItemize}

마커 비트는 페이로드 특정적인 의미를 가지며, H.\+264 비디오의 경우\+:
\begin{DoxyItemize}
\item 프레임의 마지막 패킷임을 나타낼 때 1로 설정
\item 프레임의 중간 패킷들에서는 0으로 설정
