\doxysubsection{Detailed Description}
TCP 연결 관리 및 RTSP 메시지 처리를 위한 싱글톤 클래스 

TCP 소켓을 생성하고 클라이언트 연결을 관리하며, RTSP 프로토콜에 따른 메시지 송수신을 처리하는 기능을 제공한다.
