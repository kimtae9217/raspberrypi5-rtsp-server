Detailed Description

미디어 프레임 캡처 및 버퍼 관리를 위한 싱글톤 클래스

순환 큐 기반의 프레임 버퍼 관리 시스템을 구현한 클래스로, 프레임 데이터를 캡처하고 버퍼에 저장하며, 필요 시 프레임 데이터를 반환하는 기능을 제공한다.
