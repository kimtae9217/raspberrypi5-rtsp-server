Detailed Description

RTCP 패킷 관리를 위한 클래스

< 1바이트 정렬로 패딩 없이 메모리에 패킷 구조체 배치

실제 구현은 RTCPPacket.hpp에 정의되어 있으며, RTCP 패킷의 생성, 관리, 전송 클래스

RTCP 패킷을 생성하고 전송하는 클래스

RTCP Sender Report 패킷의 구조를 정의하고, 네트워크 바이트 오더 변환과 UDP 소켓을 통한 전송을 처리한다.
