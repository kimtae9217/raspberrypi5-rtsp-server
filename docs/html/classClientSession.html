Detailed Description

RTSP 클라이언트 세션을 관리하는 클래스

각 클라이언트 연결에 대한 세션 정보를 저장하고 관리하는 클래스로, 클라이언트의 연결 정보, 스트리밍 상태, 미디어 전송을 위한 소켓 정보 등을 관리한다.
