Detailed Description

RTP/RTCP를 위한 UDP 소켓 관리 클래스

클라이언트와의 미디어 스트리밍을 위한 UDP 소켓을 관리하며, RTP/RTCP 프로토콜에 필요한 소켓과 주소 정보를 제공한다.

bool 소켓 생성 성공 여부

클라이언트의 RTP/RTCP 포트로 연결할 UDP 소켓을 생성

UDP 소켓 생성 과정:

  1. RTP와 RTCP 소켓 각각 생성
  2. 주소 구조체 초기화 및 설정
    • AF_INET: IPv4 프로토콜
    • SOCK_DGRAM: UDP 소켓 타입
  3. 클라이언트의 IP 주소와 포트 번호로 주소 구조체 설정
